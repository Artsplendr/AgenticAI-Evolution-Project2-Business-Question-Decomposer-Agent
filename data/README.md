# sample_events.csv

Daily aggregated metrics with optional segments.

Columns:
- date (YYYY-MM-DD)
- sessions (int)
- conversions (int)
- step_view_product (int)
- step_add_to_cart (int)
- step_checkout (int)
- step_purchase (int)
- device (str) e.g. desktop/mobile
- channel (str) e.g. paid_search/paid_social/email/organic
- country (str) e.g. US/DE/UK
